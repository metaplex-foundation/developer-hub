## Common Use Cases

### Asset Discovery
- [Find Who Holds a Specific Token](/dev-tools/das-api/guides/find-token-holders) - Discover all wallets holding a particular token
- [Get All Tokens in a Collection](/dev-tools/das-api/guides/get-collection-nfts) - Retrieve all assets belonging to a specific collection

### Wallet Analysis
- [Get All Tokens in a Wallet](/dev-tools/das-api/guides/get-wallet-tokens) - See everything a wallet owns
- [Get Fungible Assets by Owner](/dev-tools/das-api/guides/get-fungible-assets) - Find all fungible tokens in a wallet
- [Get NFTs by Owner](/dev-tools/das-api/guides/get-nfts-by-owner) - Retrieve only non-fungible tokens from a wallet

### Advanced Queries
- [Search Assets by Multiple Criteria](/dev-tools/das-api/guides/search-by-criteria) - Combine multiple filters to find specific assets
- [Get Assets by Owner and Collection](/dev-tools/das-api/guides/owner-and-collection) - Find tokens from a specific collection owned by a wallet
- [Find Compressed NFTs](/dev-tools/das-api/guides/find-compressed-nfts) - Discover compressed NFTs and their proofs

### Collection Management
- [Analyze Collection Statistics](/dev-tools/das-api/guides/collection-statistics) - Get insights about collection distribution and ownership
