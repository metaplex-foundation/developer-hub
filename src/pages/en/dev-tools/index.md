## Metaplex Developer Tools

Metaplex provides a comprehensive suite of developer tools to help you build, test, and deploy digital asset applications on Solana. From command-line utilities to full-featured SDKs, we've got you covered.

### SDKs & Libraries

- **[Umi](/dev-tools/umi)** - Modern TypeScript framework for Solana development with plugin architecture
- **[DAS API](/dev-tools/das-api)** - Digital Asset Standard API for querying NFTs and compressed assets at scale

### Command Line Tools

- **[CLI](/dev-tools/cli)** - Command-line interface for common Metaplex operations
- **[Shank](/dev-tools/shank)** - Generate IDLs from Rust programs for client generation

### Testing & Development

- **[Amman](/dev-tools/amman)** - Local validator toolkit for testing Solana programs

### Getting Started

New to Metaplex development? Here's the recommended path:

1. **Install the CLI** - Get up and running quickly with `npm install -g @metaplex-foundation/cli`
2. **Learn Umi** - Our TypeScript framework makes Solana development straightforward
3. **Use DAS API** - Query NFTs efficiently without running your own indexer
