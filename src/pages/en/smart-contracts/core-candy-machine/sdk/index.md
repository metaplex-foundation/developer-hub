## Summary

The [Core Candy Machine](/smart-contracts/core-candy-machine) program provides SDKs for JavaScript and Rust, enabling developers to create, configure, and mint from Candy Machines on Solana in their preferred language. {% .lead %}

- The JavaScript SDK uses the [Umi framework](/dev-tools/umi) and is available as an npm package
- The Rust SDK ships as two crates: one for the Candy Machine core and one for Candy Guards
- Both SDKs support the full Core Candy Machine feature set including guards and minting
