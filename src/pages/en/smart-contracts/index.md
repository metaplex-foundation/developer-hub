## Metaplex Smart Contracts

Metaplex provides battle-tested smart contracts (called "programs" on Solana) that power the majority of NFTs and digital assets on the network. Our programs handle everything from basic token metadata to complex NFT collections and token launches.

### Core Programs

- **[Core](/smart-contracts/core)** - Next-generation NFT standard with single-account design, reducing costs by up to 80%
- **[Token Metadata](/smart-contracts/token-metadata)** - The original NFT metadata standard, powering billions of assets on Solana
- **[Genesis](/smart-contracts/genesis)** - Token generation events (TGE), fair launches, and token sales infrastructure

### Collection & Distribution

- **[Core Candy Machine](/smart-contracts/core-candy-machine)** - NFT minting machine for Core assets with configurable guards
- **[Candy Machine](/smart-contracts/candy-machine)** - Classic NFT drop program with guards for allowlists, payments, and limits
- **[Bubblegum](/smart-contracts/bubblegum-v2)** - Compressed NFTs using state compression for massive scale at minimal cost

### Advanced Features

- **[MPL-Hybrid](/smart-contracts/mpl-hybrid)** - Swap between NFTs and fungible tokens
- **[Inscriptions](/smart-contracts/inscription)** - On-chain data storage for fully on-chain NFTs
- **[Hydra](/smart-contracts/hydra)** - Fanout wallets for splitting payments between multiple shareholders
- **[Token Auth Rules](/smart-contracts/token-auth-rules)** - Custom authorization rules for programmable NFTs
