## Summary

[Core Candy Machine](/ja/smart-contracts/core-candy-machine)のAPIリファレンスは、JavaScript SDK（TypeDoc経由）およびRustクレート（docs.rs経由）の自動生成ドキュメントを提供します。 {% .lead %}

- JavaScript SDKドキュメントは、すべての命令、アカウント、型をカバー
- Rustドキュメントは、Candy Machine CoreクレートとCandy Guardクレートに分かれている
- ソースコードは直接調査のためにGitHubで利用可能

## Quick Reference

| リソース | リンク |
|----------|------|
| JS SDK TypeDoc | [mpl-core-candy-machine.typedoc.metaplex.com](https://mpl-core-candy-machine.typedoc.metaplex.com/) |
| Rust Candy Machine Core | [docs.rs/mpl-core-candy-machine-core](https://docs.rs/mpl-core-candy-machine-core/) |
| Rust Candy Guard | [docs.rs/mpl-core-candy-guard](https://docs.rs/mpl-core-candy-guard/) |
| ソースコード | [GitHub](https://github.com/metaplex-foundation/mpl-core-candy-machine) |

## JavaScript SDKリファレンス

[TypeDocドキュメント](https://mpl-core-candy-machine.typedoc.metaplex.com/)は、`@metaplex-foundation/mpl-core-candy-machine`パッケージのすべてのエクスポートされた関数、型、アカウントをカバーしています。

## Rustクレートリファレンス

Rust APIドキュメントは2つのクレートに分かれています:

- [mpl-core-candy-machine-core](https://docs.rs/mpl-core-candy-machine-core/) -- Candy Machineの作成、設定、アセット管理をカバー
- [mpl-core-candy-guard](https://docs.rs/mpl-core-candy-guard/) -- ガードの作成、設定、ミント条件の適用をカバー
