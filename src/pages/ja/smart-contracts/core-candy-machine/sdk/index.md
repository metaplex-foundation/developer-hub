## Summary

[Core Candy Machine](/ja/smart-contracts/core-candy-machine)プログラムは、JavaScriptおよびRust用のSDKを提供しており、開発者はお好みの言語でSolana上のCandy Machineを作成、設定、ミントできます。 {% .lead %}

- JavaScript SDKは[Umiフレームワーク](/ja/dev-tools/umi)を使用し、npmパッケージとして利用可能
- Rust SDKは2つのクレートとして提供：Candy Machineコア用とCandy Guard用
- 両方のSDKは、ガードとミントを含むCore Candy Machineの完全な機能セットをサポート
