## エスクロー設定の取得

エスクロー設定アカウントを取得するには、`fetchEscrowV1`関数を使用できます。

```ts
const escrowAddress = publicKey('11111111111111111111111111111111')

const escrow = await fetchEscrowV1(umi, escrowAddress)
```
