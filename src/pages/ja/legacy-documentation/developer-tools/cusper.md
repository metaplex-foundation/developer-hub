# Cusper

Cusperは、Solanaプログラムのログまたはエラーコードから**Cus**tom **P**rogram **Er**rors(カスタムプログラムエラー)を解決します。[Solita](solita)によってプログラムエラーの処理を大幅に簡単にするために使用されます。
