## Metaplex開発ツール

Metaplexは、Solana上でデジタルアセットアプリケーションを構築、テスト、デプロイするための包括的な開発者ツールスイートを提供します。コマンドラインユーティリティからフル機能のSDKまで、すべてをカバーしています。

### SDK＆ライブラリ

- **[Umi](/ja/dev-tools/umi)** - プラグインアーキテクチャを備えたSolana開発用モダンTypeScriptフレームワーク
- **[DAS API](/ja/dev-tools/das-api)** - NFTと圧縮アセットを大規模にクエリするためのDigital Asset Standard API

### コマンドラインツール

- **[CLI](/ja/dev-tools/cli)** - 一般的なMetaplex操作用のコマンドラインインターフェース
- **[Shank](/ja/dev-tools/shank)** - クライアント生成のためにRustプログラムからIDLを生成

### テスト＆開発

- **[Amman](/ja/dev-tools/amman)** - Solanaプログラムをテストするためのローカルバリデータツールキット

### はじめに

Metaplex開発が初めてですか？推奨パスはこちら：

1. **CLIをインストール** - `npm install -g @metaplex-foundation/cli`ですぐに始められます
2. **Umiを学ぶ** - TypeScriptフレームワークでSolana開発が簡単に
3. **DAS APIを使う** - 独自のインデクサーを実行せずにNFTを効率的にクエリ

### なぜこれらのツールなのか？

- **TypeScriptファースト** - 完全な型安全性を備えたモダンな開発者体験
- **モジュラー設計** - 必要なものだけを使用
- **アクティブなメンテナンス** - 定期的なアップデートと改善
- **コミュニティサポート** - Metaplexで構築する数千人の開発者に参加
