## Metaplex 스마트 컨트랙트

Metaplex는 네트워크에서 대부분의 NFT와 디지털 자산을 지원하는 검증된 스마트 컨트랙트(Solana에서는 "프로그램"이라고 함)를 제공합니다. 기본 토큰 메타데이터부터 복잡한 NFT 컬렉션과 토큰 런칭까지 모든 것을 처리합니다.

### 핵심 프로그램

- **[Core](/ko/smart-contracts/core)** - 단일 계정 설계의 차세대 NFT 표준, 비용 최대 80% 절감
- **[Token Metadata](/ko/smart-contracts/token-metadata)** - Solana에서 수십억 자산을 지원하는 원조 NFT 메타데이터 표준
- **[Genesis](/ko/smart-contracts/genesis)** - 토큰 생성 이벤트(TGE), 페어 런칭, 토큰 세일 인프라

### 컬렉션 & 배포

- **[Core Candy Machine](/ko/smart-contracts/core-candy-machine)** - 구성 가능한 가드가 있는 Core 자산용 NFT 민팅 머신
- **[Candy Machine](/ko/smart-contracts/candy-machine)** - 허용 목록, 결제, 제한을 위한 가드가 있는 클래식 NFT 드롭 프로그램
- **[Bubblegum](/ko/smart-contracts/bubblegum-v2)** - 최소 비용으로 대규모 확장을 위한 상태 압축을 사용하는 압축 NFT

### 고급 기능

- **[MPL-Hybrid](/ko/smart-contracts/mpl-hybrid)** - NFT와 대체 가능 토큰 간 스왑
- **[Inscriptions](/ko/smart-contracts/inscription)** - 완전 온체인 NFT를 위한 온체인 데이터 스토리지
- **[Hydra](/ko/smart-contracts/hydra)** - 여러 주주 간 결제 분할을 위한 팬아웃 지갑
- **[Token Auth Rules](/ko/smart-contracts/token-auth-rules)** - 프로그래머블 NFT를 위한 커스텀 인증 규칙

### 왜 Metaplex 프로그램인가?

- **검증됨** - 수십억 달러의 디지털 자산 보호
- **오픈 소스** - 완전히 감사되고 투명함
- **조합 가능** - 프로그램이 원활하게 함께 작동
- **잘 문서화됨** - 포괄적인 가이드와 API 레퍼런스
- **활발한 개발** - 지속적인 개선과 새로운 기능
