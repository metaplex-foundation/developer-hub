## 요약

[Core Candy Machine](/ko/smart-contracts/core-candy-machine) API 레퍼런스는 JavaScript SDK(TypeDoc을 통해)와 Rust 크레이트(docs.rs를 통해)에 대한 자동 생성 문서를 제공합니다. {% .lead %}

- JavaScript SDK 문서는 모든 명령, 계정, 타입을 포함합니다
- Rust 문서는 Candy Machine Core 크레이트와 Candy Guard 크레이트로 나뉩니다
- 소스 코드는 직접 검사를 위해 GitHub에서 사용할 수 있습니다

## JavaScript SDK 레퍼런스

[TypeDoc 문서](https://mpl-core-candy-machine.typedoc.metaplex.com/)는 `@metaplex-foundation/mpl-core-candy-machine` 패키지의 모든 내보낸 함수, 타입, 계정을 다룹니다.

## Rust 크레이트 레퍼런스

Rust API 문서는 두 개의 크레이트로 나뉩니다:

- [mpl-core-candy-machine-core](https://docs.rs/mpl-core-candy-machine-core/) -- Candy Machine 생성, 구성, 에셋 관리를 다룹니다
- [mpl-core-candy-guard](https://docs.rs/mpl-core-candy-guard/) -- 가드 생성, 구성, 민팅 조건 적용을 다룹니다
