## 요약

[Core Candy Machine](/ko/smart-contracts/core-candy-machine) 프로그램은 JavaScript 및 Rust용 SDK를 제공하여, 개발자가 선호하는 언어로 Solana에서 Candy Machine을 생성, 구성, 민팅할 수 있게 합니다. {% .lead %}

- JavaScript SDK는 [Umi 프레임워크](/ko/dev-tools/umi)를 사용하며 npm 패키지로 제공됩니다
- Rust SDK는 두 개의 크레이트로 제공됩니다: Candy Machine Core용 하나와 Candy Guard용 하나
- 두 SDK 모두 가드 및 민팅을 포함한 전체 Core Candy Machine 기능 세트를 지원합니다
