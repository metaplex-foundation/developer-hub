## Metaplex 개발 도구

Metaplex는 Solana에서 디지털 자산 애플리케이션을 구축, 테스트, 배포하는 데 도움이 되는 포괄적인 개발자 도구 모음을 제공합니다. 커맨드라인 유틸리티부터 완전한 기능의 SDK까지 모두 갖추고 있습니다.

### SDK & 라이브러리

- **[Umi](/ko/dev-tools/umi)** - 플러그인 아키텍처를 갖춘 Solana 개발용 모던 TypeScript 프레임워크
- **[DAS API](/ko/dev-tools/das-api)** - NFT와 압축 자산을 대규모로 쿼리하기 위한 Digital Asset Standard API

### 커맨드라인 도구

- **[CLI](/ko/dev-tools/cli)** - 일반적인 Metaplex 작업을 위한 커맨드라인 인터페이스
- **[Shank](/ko/dev-tools/shank)** - 클라이언트 생성을 위해 Rust 프로그램에서 IDL 생성

### 테스트 & 개발

- **[Amman](/ko/dev-tools/amman)** - Solana 프로그램 테스트를 위한 로컬 밸리데이터 툴킷

### 시작하기

Metaplex 개발이 처음이신가요? 권장 경로는 다음과 같습니다:

1. **CLI 설치** - `npm install -g @metaplex-foundation/cli`로 빠르게 시작
2. **Umi 배우기** - TypeScript 프레임워크로 Solana 개발이 간단해집니다
3. **DAS API 사용** - 자체 인덱서 없이 NFT를 효율적으로 쿼리

### 왜 이 도구들인가?

- **TypeScript 우선** - 완전한 타입 안전성을 갖춘 현대적 개발자 경험
- **모듈식 설계** - 필요한 것만 사용
- **활발한 유지보수** - 정기적인 업데이트와 개선
- **커뮤니티 지원** - Metaplex로 구축하는 수천 명의 개발자와 함께하세요
