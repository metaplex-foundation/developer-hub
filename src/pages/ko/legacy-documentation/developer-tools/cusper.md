# Cusper

Cusper는 Solana 프로그램 로그 또는 오류 코드에서 **사용자 정의 프로그램 오류**를 해결합니다. [Solita](solita)에서 프로그램 오류 처리를 훨씬 쉽게 만들기 위해 사용됩니다.
