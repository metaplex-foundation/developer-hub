欢迎使用 DAS API 指南！这些实用示例将帮助您解决在 Solana 上处理数字资产时的常见用例。

## 常见用例

### 资产发现

- [查找持有特定代币的用户](/zh/dev-tools/das-api/guides/find-token-holders) - 发现持有特定代币的所有钱包
- [获取集合中的所有代币](/zh/dev-tools/das-api/guides/get-collection-nfts) - 检索属于特定集合的所有资产

### 钱包分析

- [获取钱包中的所有代币](/zh/dev-tools/das-api/guides/get-wallet-tokens) - 查看钱包拥有的所有内容
- [按所有者获取同质化资产](/zh/dev-tools/das-api/guides/get-fungible-assets) - 查找钱包中的所有同质化代币
- [按所有者获取 NFT](/zh/dev-tools/das-api/guides/get-nfts-by-owner) - 仅从钱包检索非同质化代币

### 高级查询

- [按多个条件搜索资产](/zh/dev-tools/das-api/guides/search-by-criteria) - 组合多个过滤器查找特定资产
- [按所有者和集合获取资产](/zh/dev-tools/das-api/guides/owner-and-collection) - 查找钱包拥有的特定集合的代币
- [查找压缩 NFT](/zh/dev-tools/das-api/guides/find-compressed-nfts) - 发现压缩 NFT 及其证明

### 集合管理

- [分析集合统计数据](/zh/dev-tools/das-api/guides/collection-statistics) - 获取有关集合分布和所有权的见解
