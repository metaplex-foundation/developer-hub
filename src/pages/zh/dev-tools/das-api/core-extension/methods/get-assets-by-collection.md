---

返回具有给定权限的所有 MPL Core 资产的信息，包括其元数据、所有者和解析的插件数据，以及从集合继承的数据。

## 代码示例

将 `<ENDPOINT>` 替换为您的个人 RPC，将 `<PublicKey>` 值替换为您要获取 NFT 的集合的公钥。

```js
import { createUmi } from '@metaplex-foundation/umi-bundle-defaults';
import { dasApi } from '@metaplex-foundation/digital-asset-standard-api';
import { das }  from '@metaplex-foundation/mpl-core-das';
import { publicKey } from '@metaplex-foundation/umi';

const umi = createUmi('<ENDPOINT>').use(dasApi());
const collection = publicKey('<PublicKey>');

const assets = await das.getAssetsByCollection(umi, { collection });
console.log(assets);
```
