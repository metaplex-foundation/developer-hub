## 如何访问 Aura 网关?

Aura 网关目前已禁用,正在进行技术审查,以解决某些错误和性能问题。

## Aura 支持哪些协议?

Aura 支持以下协议的最新 DAS 方法:
- MPL Token Metadata (NFT/pNFT)
- MPL Core (Core Assets/Core Collections)
- MPL Bubblegum (cNFTs)
- Fungible SPL Tokens
- SPL Token22
