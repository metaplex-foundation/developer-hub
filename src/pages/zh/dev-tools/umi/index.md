一个用于 JavaScript 客户端的 Solana 框架。

Umi 是一个用于构建和使用 Solana 程序 JavaScript 客户端的模块化框架。它提供了一个零依赖库，定义了一组库可以依赖的核心接口，而不绑定到特定实现。然后由最终用户选择最适合其需求的实现。Umi 还提供了一组默认实现和捆绑包，可以开箱即用，让开发者快速上手。{% .lead %}
