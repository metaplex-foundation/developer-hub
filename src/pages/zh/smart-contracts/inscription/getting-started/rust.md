如果您是Rust开发者，您也可以使用Rust客户端SDK与Token Metadata程序交互。Metaplex提供了一个专用的Rust客户端crate，这是一个具有最小依赖项的轻量级crate。

要开始，您需要将`mpl-inscription`依赖项添加到您的项目中。从项目根文件夹的终端：

```
cargo add mpl-inscription
```

这将在项目的依赖项列表中添加该crate的最新版本。
