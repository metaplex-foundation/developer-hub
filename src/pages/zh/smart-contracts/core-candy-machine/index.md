Metaplex 协议的 **Candy Machine** 是 Solana 上用于公平 NFT 集合发布的领先铸造和分发程序。随着 `Metaplex Core 协议`简化了 Solana 上的 NFT 流程,Candy Machine 的 Core 版本也应运而生。正如其名称所暗示的那样,您可以将 Candy Machine 视为一个临时结构,首先由创作者加载,然后由买家卸载。它允许创作者以安全和可定制的方式将其数字资产上链。{% .lead %}

这个名称指的是通过机械曲柄分发糖果的自动售货机。在这种情况下,糖果是 NFT,支付方式是 SOL 或 SPL 代币。
