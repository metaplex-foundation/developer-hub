## 概要

[Core Candy Machine](/zh/smart-contracts/core-candy-machine) 程序提供 JavaScript 和 Rust SDK，使开发者能够使用其首选语言在 Solana 上创建、配置和铸造 Candy Machine。 {% .lead %}

- JavaScript SDK 使用 [Umi 框架](/zh/dev-tools/umi)，以 npm 包形式提供
- Rust SDK 以两个 crate 发布：一个用于 Candy Machine 核心，一个用于 Candy Guard
- 两个 SDK 都支持完整的 Core Candy Machine 功能集，包括守卫和铸造
