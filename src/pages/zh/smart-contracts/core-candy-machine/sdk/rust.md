## Core Candy Machine

Core Candy Machine 是 Candymachine 的核心,允许初始化和填充 Core Candy Machine

### 安装

MPL-Core Rust SDK 可用于脚本/桌面/移动应用程序以及 Solana 链上程序。

```rust
cargo add mpl-core-candy-machine-core
```
