## 概要

[Core Candy Machine](/zh/smart-contracts/core-candy-machine) API 参考提供了 JavaScript SDK（通过 TypeDoc）和 Rust crate（通过 docs.rs）的自动生成文档。 {% .lead %}

- JavaScript SDK 文档涵盖所有指令、账户和类型
- Rust 文档分为 Candy Machine Core crate 和 Candy Guard crate
- 源代码可在 GitHub 上直接查看

## JavaScript SDK 参考

[TypeDoc 文档](https://mpl-core-candy-machine.typedoc.metaplex.com/)涵盖了 `@metaplex-foundation/mpl-core-candy-machine` 包中所有导出的函数、类型和账户。

## Rust Crate 参考

Rust API 文档分为两个 crate：

- [mpl-core-candy-machine-core](https://docs.rs/mpl-core-candy-machine-core/) -- 涵盖 Candy Machine 的创建、配置和资产管理
- [mpl-core-candy-guard](https://docs.rs/mpl-core-candy-guard/) -- 涵盖守卫的创建、配置和铸造条件强制执行
