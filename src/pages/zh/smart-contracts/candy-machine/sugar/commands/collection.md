您可以使用 `collection` 命令手动设置 Candy Machine 的 collection NFT。您只能在从中铸造任何 NFT 之前修改 collection。

您可以通过运行以下命令设置预先存在的 collection NFT：

```
sugar collection set <COLLECTION MINT>
```

其中 `<COLLECTION MINT>` 是 collection NFT 铸币账户的公钥。
