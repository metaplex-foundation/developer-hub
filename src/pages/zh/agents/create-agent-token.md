代理代币是从代理的链上钱包发行的代币。

首先，代理需要[在 Solana 上向 Metaplex 注册](/agents/register-agent)，这将创建公开身份和钱包。

然后，代理通过 Metaplex API、SDK 或 CLI 使用 [Metaplex Genesis 协议](/smart-contracts/genesis)发行代币。
