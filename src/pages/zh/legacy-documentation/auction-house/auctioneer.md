定时拍卖拍卖人是一个拍卖人实现,它在拍卖行之上添加了英式拍卖规则。

# 列表配置
列表配置结构用于存储附加到每个列表的所有与功能相关的参数。所有功能都可以按每个列表进行配置。

# 功能
* 带有开始和结束日期的英式拍卖
* 最高出价跟踪
* 保留价格(最低出价)
* 最低出价增量 - 要求新出价者的出价比当前出价高一定金额
* 自动时间延长 - 在接近拍卖结束时(此期间可自定义)出价将延长拍卖结束日期一个可配置的时间量
* 防止最高出价者取消 - 最高出价者将无法取消他们的出价,如果他们赢得拍卖,则需要购买 NFT*

{% callout type="warning" %}

*由于拍卖行的无托管性质,目前无法防止用户从钱包中转移列出的代币并使其出价无效。

{% /callout %}
